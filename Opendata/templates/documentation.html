General Transit Feed Specification - GTFS

The Delhi Bus data is made public according to GTFS. GTFS defines a common format for public transportation schedules and associated geographic information.


Static Data

The static timetable of Delhi Buses can be accessed using the following files:

  • agency.txt
  • calendar.txt
  • stops.txt
  • routes.txt
  • trips.txt
  • stop_times.txt

The format of the files can be found here.


Warnings
The arrival and departure times of buses as mentioned in stop_times.txt is not accurate and is a rough estimate generated by assuming a constant speed of travel.

Real Time Data

Authorised users can request for real-time data at /api/realtime/VehiclePositions.pb?key=YOUR_PRIVATE_KEY
